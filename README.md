# FindandSolve — Demo Website

Das erste vollautomatisierte Versicherungsmaklerbüro Deutschlands.

**16 Features · 9 APIs · 4 Devices**
Vom Makler, für den Makler. Gebaut in Rockenhausen, Powered by AI.

## Seiten

| Seite | Datei | Beschreibung |
|-------|-------|--------------|
| Hauptseite | `index.html` | Landingpage mit Feature-Übersicht, Architektur-Diagramm und Pricing |
| Kundenportal | `kundenportal.html` | Self-Service-Portal für Versicherungskunden |
| PDF-Splitter | `pdf-splitter.html` | KI-gestütztes Aufteilen und Zuordnen von Dokumenten |
| KFZ-Optimierer | `kfz-optimierer.html` | Automatische Tarifanalyse & Wechselempfehlungen für KFZ-Bestände |
| KI-Regeln | `ki-regeln.html` | 3-Layer-Regelwerk: Global, Sparten, Kunden |

## Lokal öffnen

Einfach `index.html` im Browser öffnen — kein Build-Prozess nötig.

Oder mit einem lokalen Server:

```bash
npx serve .
```

## Tech-Stack

- Pure HTML/CSS/JavaScript (kein Framework)
- Responsive Design
- Dark Theme mit Glassmorphism-Effekten
- Animationen via CSS & Vanilla JS

## Kontext

Diese Demo-Website zeigt die Features der FindandSolve-Plattform.
Die eigentliche Plattform besteht aus einem Next.js-Frontend und einem FastAPI-Backend.
